# Atlântico Studio — Premium Final

Site premium responsivo pronto para GitHub Pages.

## Como publicar
1. Apague os arquivos antigos do repositório.
2. Suba todos os arquivos deste ZIP na raiz do repositório.
3. Vá em Settings > Pages.
4. Source: Deploy from branch.
5. Branch: main.
6. Folder: /root.
7. Aguarde 1 a 3 minutos.
